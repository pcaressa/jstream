{\footnotesize\ttfamily \mbox{\hyperlink{jstream_8h_a519b778285205c898b57839ab698552c}{jstream\+\_\+t}} jstream (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{jstream_8h_a076b8f0d0ff404f7d0f03adf5e92b214}{jstream\+\_\+param\+\_\+t}}}]{p }\end{DoxyParamCaption})}

Parse a json stream\+: it the get field of the structure $\ast$p to be assigne to a function (pointer) returning the next character read from the stream, or a negative value in case of end or error. Return the following values inside the structure $\ast$p\+: obj = pointer to the memory area containing data size = length of such area in bytes error = error code (0 meaning \char`\"{}no error\char`\"{}) clast = last item got from the stream and that follows the parsed json value. The address obj is also returned as value from the function (in case of error it is NULL). Warning\+: it is the caller responsibility to deallocate p-\/$>$obj once it is no longer needed, via free(p-\/$>$obj). Here is the call graph for this function\+:
